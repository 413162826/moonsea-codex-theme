# 月海官网

月海 Codex 主题的 Sites 站点，包含公开主题墙、安装下载入口、匿名使用统计接口和仅管理员可见的数据页。

## 隐私边界

- 使用统计默认关闭，只有用户在月海助手中主动开启后才会上报。
- 仅记录随机安装标识、月海版本、操作系统、架构和最近活跃时间。
- 不读取或上传 Codex 账号、邮箱、提示词、项目名称、文件路径和壁纸内容。
- `/admin` 使用 ChatGPT 登录，并在服务端校验管理员邮箱白名单。

## 本地开发

```bash
npm ci
npm run dev
npm test
```

修改 `db/schema.ts` 后运行 `npm run db:generate`，生成并检查 D1 迁移再发布。
