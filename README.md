# Codex 月海主题

给 Codex 加上月海壁纸、透明面板和更清晰的按钮。官方客户端不会被修改，安装后会多出一个“Codex 月海版”。

## 安装

先关闭 Codex，然后到 [Releases](https://github.com/413162826/moonsea-codex-theme/releases/latest) 下载最新版并解压。

### Windows

双击 `Install-Moonsea-Windows.cmd`。

### macOS

根据 Mac 芯片下载 Apple Silicon 或 Intel 版本，然后右键打开 `Install-Moonsea-macOS.command`。

不知道芯片类型？点左上角苹果图标 →“关于本机”：显示 M1/M2/M3/M4/M5 就选 Apple Silicon，显示 Intel 就选 Intel。

## 使用

从桌面打开“Codex 月海版”。右下角可以调透明度、更换壁纸。

## 更新

下载新版本，再运行一次安装脚本。登录、壁纸和设置都会保留。

## 卸载

运行对应系统的 `Uninstall` 脚本。卸载不会影响官方 Codex。

> 官方 Codex 更新后，再运行一次安装脚本即可。
